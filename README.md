### Built With

- Semantic HTML5
- CSS Grid and Flexbox
- Mobile-first workflow

### Features

- Responsive layout across mobile, tablet, and desktop
- Accessible markup and logical content order

### Getting Started

Clone and open the `index.html` in your browser:

```bash
git clone https://github.com/LucasMontecino/bento-grid-frontend-mentor.git
cd bento-grid-main
# Open index.html in your browser
```

### Project Structure

- `index.html`: markup
- `styles.css`: styles
- `assets/`: images
- `design/`: reference designs (desktop/mobile JPG)
- `style-guide.md`: colors, typography, spacing
